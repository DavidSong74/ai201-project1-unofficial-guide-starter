User_237267: For those with MacOS (and iOS), I've been using Arc (https://arc.net/) for half a year now and it's been a better alternative to Chrome for me. Has everything Chrome have (because it's built on Chrominium), but much nicer UI and constant development. Some features I find useful are
- Command Bar to do everything, from splitting tabs, opening a new tab, search directly on ChatGPT, YouTube, Figma, etc
- Better hot keys, and you can customize how you switch from one tab to another, copy current link, share quotes instantly, etc
- Little Arc window, so you can watch YouTube, meetings, etc on a small floating screen when you're in another tab
- Better organization of tabs and spaces. In Chrome, usually you keep them cluttered on top and can't seen what you're working on. In Arc you have tabs on the left-hand side (or where you customize it to be) and usually hidden, you have more space to work on and can switch tabs more easily if you know hot keys. You can also have spaces for different work contexts and projects
- Preview of pinned tabs: When I pin Google Calendar, I can see my timeline when I hover on it. Applicable for Notion, Figma, and other popular websites I believe
- More small features: customize the web page background, rename tabs/downloads for better comprehension, summarize web articles, etc

If you have tried SigmaOS, I'd argue Arc is probably better and more configuration. If you're into security though, Arc's not the go-to web browser afaik (maybe Brave or whatnot is better)
